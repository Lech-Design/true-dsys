# Decorators

<br />
## What are Decorators?

Decorators pre-populate existing MudBlazor components with a
set of predefined properties and CSS utility classes.

All decorators are prefixed with the word **True**, for example **_TruePaper_** or **_TrueDrawerHeader_**.

Be defining a set of reusable decorator components, we can maintain greater consistency throught the app experience.
