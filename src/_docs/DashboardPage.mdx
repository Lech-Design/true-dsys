# True Dash Page

This pattern provides the overall page structure for the True Dashboard.

This page is constructed with the following component hierarchy:

```js
<TrueDashSection>
    <MudItem>
        <TrueWidget>
            <!-- Optional Header -->
            <TrueWidgetHeader>
                <!-- Optional Interactive actions  -->
            </TrueWidgetHeader>
            @Place your main content here
        </TrueWidget>
    </MudItem>
</TrueDashSection>
```
